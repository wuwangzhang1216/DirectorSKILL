## Rules

- Shot number should follow story order, not production order.
- Beat/story function must be explicit. If it is empty, the shot probably should not exist.
- Blocking must contain a start and end state.
- Camera movement should be one dominant move.
- Continuity anchors should mention face/costume/location/props/light when relevant.
